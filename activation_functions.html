Implementation of different activation functions¶

This is an example implementation of activation functions like Sigmoid, Hyperbolic Tangent Function (tanh) and Recitfied Linear Unit (ReLU).

Please bear in mind that these activation functions are to be applied to the outputs of hidden layers. Please apply Softmax to the output layer if it is a classification problem or a linear function otherwise.¶

Sigmoid and it's derivative¶

It takes a number and squashes it between 0 and 1. It can be thought of as an on/off state where 0 would mean that the neuron is off i.e. the neuron won't produce an output and 1 would mean that the neuron is on i.e. it will generate some output. However, SIgmoid suffers from vanishing gradient problem. Also, the output is not 0 centered.

Vanishing Gradient¶

When a neuron's output saturates either close to 0 or 1, the gradients at these regions is close to 0. During back propagation, if the local gradient is too small, it will make the gradient slowly vanish.

0 Centering Problem¶

If the output is not 0 centered, the value of the function will be positive and that would make the weights of the gradient either all positive or all negative. This makes gradients go too far which makes optimization harder.

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def sigmoid_derivative(x):
    return x * (1 - x)

Hyperbolic Tangent Function (tanh) and it's derivative¶

It squashes the value between -1 and 1. Thus, the output is 0 centered which makes optimization easier. However, just like the sigmoid, it suffers from the vanishing gradient problem.

In [3]:
def tanh(x):
    return np.tanh(x)

def tanh_derivative(x):
    return 1 - (x ** 2)

Recitfied Linear Unit (ReLU) and it's derivative¶

def relu(x):
    for i in range(0, len(x)):
        for k in range(0, len(x[i])):
            if x[i][k] > 0:
                pass  # do nothing since it would be effectively replacing x with x
            else:
                x[i][k] = 0
    return x

def relu_derivative(x):
    for i in range(0, len(x)):        
        for k in range(len(x[i])):            
            if x[i][k] > 0:                
                x[i][k] = 1
            else:
                x[i][k] = 0
    return x
